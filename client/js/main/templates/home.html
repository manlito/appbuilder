Simple App Design

The editor uses one page, and there is no flow diagrams to do. The philosophy is that should be simple to use, at the sacrifice of features.

One Click Publish

Just provide your API Keys from Heroku and MongoLab and you're ready to start using it. Recommendation: use your free quotas. For a small number of users it should be free.

Extensible

Source code written for NodeJS is provided, for both this website and the generated apps.

Project Vision

The idea is to evolve this app into a BPM for the small company. In addition, the target platform is mobile, and some fields, such as GPS or CameraPicture will be added and supported through an App to be done.
